# Sample go_server
This is a sample README for myserver

## To build 
`go build` 

## To run 
`./go_server <PORT>`

## To test
In a browser open URL:
 `http://<IP Addr>:<PORT>/v1` --> prints `hello`

 `http://<IP Addr>:<PORT>/test` --> prints hostname

 `http://<IP Addr>:<PORT>/gen` --> creates a new certificate for host
